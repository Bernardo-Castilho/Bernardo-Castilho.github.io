Wijmo and webpack

This page includes a single script file that contains a bundle with all the Wijmo CSS and JS files needed by the sample.

The bundle file was created using the webpack module bundler.

These are the steps we used to create the bundle file:

  1. Install node.js
  2. Install webpack: npm install webpack -g
  3. Install the webpack loaders we need: npm install script-loader style-loader css-loader --save
  4. Create a main.js file that requires all the files needed by the app.
  5. Create a webpack.config.js file that contains webpack configuration commands.
  6. Use webpack to create the bundle: webpack.
  7. Add the static/bundle.js file as a script tag on the page.
